### Vim
```
" Bazel
" Add maktaba, glaive and bazel to the runtimepath.
" (maktaba and glaive must be installed before it can be used.)
Plug 'google/vim-maktaba'
Plug 'google/vim-glaive'
Plug 'bazelbuild/vim-bazel'
```
